# Claude Plan Reviewer

A full-featured, independent Single Page Application (SPA) that mimics the GitLab Merge Request review experience for reviewing plans generated by Claude.

## Features

- **Batch Review**: Create multiple draft comments before submitting.
- **Inline Comments**: Attach comments to specific text selections.
- **Persistence**: Reviews are saved to the local file system (`.reviews/` directory).
- **Async Workflow**: Decoupled request and retrieve steps.

## Tech Stack

- **Frontend**: Vue 3, Rsbuild, TailwindCSS
- **Backend**: Node.js, Express, MCP SDK

## Usage

### MCP Tools

- `request_human_review`: Starts a review session.
- `get_review_result`: Retrieves the review comments.

### API

- `GET /api/reviews/:id`: Get review state.
- `POST /api/reviews/:id/comments`: Create draft.
- `PUT /api/reviews/:id/comments/:commentId`: Update draft.
- `DELETE /api/reviews/:id/comments/:commentId`: Delete draft.
- `POST /api/reviews/:id/submit`: Submit review.
